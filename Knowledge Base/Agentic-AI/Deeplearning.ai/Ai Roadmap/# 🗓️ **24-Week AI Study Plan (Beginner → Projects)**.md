

---

## 📍 Month 1: Foundations (Weeks 1–4)

**Week 1: Python Basics**

- Learn variables, loops, functions, lists, dicts.
- Practice on HackerRank / LeetCode (easy problems).

**Week 2: Python for Data Science**

- Learn NumPy (arrays, matrix operations).
- Learn Pandas (data cleaning, analysis).
- Learn Matplotlib/Seaborn (visualization).

**Week 3: Math for AI – Part 1**

- Linear Algebra → vectors, dot product, matrix multiplication.
- Probability basics → mean, variance, conditional probability.

**Week 4: Math for AI – Part 2**

- Statistics → distributions, z-score, hypothesis testing.
- Calculus → derivatives, gradient basics.
- Mini-project: Analyze a dataset (Kaggle Titanic survival).

---

## 📍 Month 2: Core Machine Learning (Weeks 5–8)

**Week 5: ML Fundamentals**

- Supervised vs unsupervised learning.
- Train-test split, evaluation metrics (accuracy, precision, recall).

**Week 6: Regression Models**

- Linear regression, logistic regression.
- Mini-project: Predict house prices.

**Week 7: Classification & Clustering**

- Decision trees, Random Forests, KNN.
- K-Means clustering, PCA (dimensionality reduction).
- Mini-project: Email spam classifier.

**Week 8: End-to-End ML**

- Build a full pipeline with scikit-learn.
- Mini-project: Customer segmentation (clustering).

---

## 📍 Month 3: Deep Learning (Weeks 9–12)

**Week 9: Neural Network Basics**

- Perceptron, activation functions (ReLU, sigmoid).
- Forward/backpropagation, gradient descent.

**Week 10: PyTorch Basics**

- Tensors, datasets, data loaders.
- Build a simple feed-forward NN.

**Week 11: CNNs (Computer Vision)**

- Convolution, pooling.
- Mini-project: MNIST digit classifier.

**Week 12: RNNs & LSTMs**

- Sequence models, vanishing gradients.
- Mini-project: Sentiment analysis with RNN/LSTM.

---

## 📍 Month 4: NLP & Transformers (Weeks 13–16)

**Week 13: NLP Basics**

- Tokenization, stopwords, Bag-of-Words, TF-IDF.
- Word embeddings (Word2Vec, GloVe).

**Week 14: Transformers & Attention**

- Self-attention, positional encoding.
- Read “Attention is All You Need” (simplified summary).

**Week 15: Hugging Face Intro**

- Install 🤗 Transformers.
- Use pre-trained models (BERT, DistilBERT).
- Mini-project: Sentiment classifier with Hugging Face.

**Week 16: Advanced NLP Applications**

- Text summarization, Q&A.
- Mini-project: News summarizer with Hugging Face.

---

## 📍 Month 5: AI Applications (Weeks 17–20)

**Week 17: LangChain Basics**

- Install LangChain.
- Use Hugging Face models in LangChain.
- Build a basic chatbot.

**Week 18: Memory & RAG**

- Add memory to chatbot.
- Vector databases (FAISS/Pinecone).
- Mini-project: Q&A chatbot over your documents.

**Week 19: Agents & Tools**

- Teach chatbot to use tools (web search, calculator).
- Mini-project: Chatbot with API integration.

**Week 20: Deployment**

- Streamlit/FastAPI for UI.
- Deploy to Hugging Face Spaces.

**Week 21: Pick Your Specialization**

- Computer Vision OR NLP/LLMs OR Generative AI.
